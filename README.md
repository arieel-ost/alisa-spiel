# 🐱 Alysas Hüpf-Abenteuer

Ein Plattform-Spiel, bei dem eine Katze durch verschiedene Levels läuft, springt, schwebt und mit Feuer & Eis schiesst.

Gebaut von **Alysa (9)** zusammen mit Claude Code.

## 🎮 Spielen

- **Pfeiltasten** — Laufen
- **⬆️ / Leertaste** — Springen (zweimal für Doppelsprung!)
- **⬇️ in der Luft** — Bodenstoss
- **X** — Feuerball / Eiskristall schiessen (wenn Power gesammelt)
- **9** — Level neu starten

## ✨ Features

- 5 Levels: Wiese · Wolken · Berggipfel · Dunkler Wald · Drachenburg
- Fragezeichen-Blöcke ❓ mit versteckten Items
- Power-Ups: 🔥 Feuer · ❄️ Eis · 🌈 Regenbogen · 🛡️ Schild · 🪶 Feder · 🍄 Pilz
- Sammeln: ⭐ Sterne · 🪙 Münzen · 💎 Diamanten · ❤️ Herzen · 👑 Kronen
- Gegner: 👾 🐛 🦇 🧙 🐉 (jagen, fliegen, schiessen)

## 🛠️ Lokal entwickeln

```bash
pnpm install
pnpm dev
```
